1. 🔧 Proceso de activación

Las licencias Pro y Empresarial se activan mediante el ID de dispositivo único de tu computadora. Al completar la compra:

  • Recibirás tu clave de licencia por correo electrónico en máximo 15 minutos.
  • Deberás ingresar la clave en el menú de activación del programa.
  • El software quedará activado permanentemente en ese dispositivo.

¿Cómo encontrar tu ID de dispositivo? Abre el programa → Menú → Activación → Ver ID de dispositivo.

2. 💻 Cambio de dispositivo

Si necesitas migrar tu licencia a otro PC (por cambio de equipo o falla de hardware), contáctame directamente con tu comprobante de compra. Se permite una transferencia por año sin costo adicional.

3. 🚫 Usos prohibidos

  • Compartir la clave de licencia con terceros.
  • Instalar el software en más dispositivos de los permitidos.
  • Realizar ingeniería inversa, descompilar o modificar el software.
  • Revender o sublicenciar el software sin autorización escrita.
  • Usar el software en sistemas de distribución masiva o nubes públicas.

4. ⏱️ Vigencia del soporte

El soporte técnico es válido durante el período indicado en cada plan (6, 12 o 24 meses) desde la fecha de compra. Una vez vencido el soporte, el software sigue funcionando pero no incluye asistencia técnica.

5. 🔄 Política de actualizaciones

Las actualizaciones se clasifican en:

  • Parches (x.x.N): correcciones de errores. Gratuitas para todos los usuarios activos.
  • Menores (x.N.x): nuevas funciones pequeñas. Incluidas durante el período de soporte.
  • Mayores (N.x.x): rediseño o grandes nuevas funciones. Pueden requerir renovación de licencia con descuento del 50% para usuarios existentes.

6. ❓ Preguntas frecuentes

¿Puedo usar el software en mi laptop y mi PC de escritorio?
El plan Pro permite hasta 2 dispositivos. El plan Básico solo 1.

¿La licencia caduca?
No. Una vez activada, la licencia es permanente. Solo el soporte técnico tiene vigencia limitada.

¿Qué pasa si reinstalo Windows?
Si el ID de dispositivo cambia tras una reinstalación, contáctame para reactivar sin costo adicional.
